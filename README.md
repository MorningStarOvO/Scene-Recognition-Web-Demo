## Scene Recognition Web Demo

### Requirement

* flask
* pillow
* pytorch
* torchvision
* opencv

<br>

### RUN

```
python app.py
```

<br>



### Test

* `static/army_base`：数据集中与「**军事**」相关的场景
* `static/web_army`：网上搜索的「**军事**」相关场景
* `static/web_building`：人民大会堂等「**建筑**」
